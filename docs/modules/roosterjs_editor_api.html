createLink

  • createLink(editor: Editor, link: string, altText?: string, displayText?: string): void
    • Defined in roosterjs-editor-api/lib/format/createLink.ts:50

    Insert a hyperlink at cursor. When there is a selection, hyperlink will be applied to the selection, otherwise a hyperlink will be inserted to the cursor position.

    Parameters

    • editor: Editor

      Editor object

    • link: string

      Link address, can be http(s), mailto, notes, file, unc, ftp, news, telnet, gopher, wais. When protocol is not specified, a best matched protocol will be predicted.

    • Optional altText: string

      Optional alt text of the link, will be shown when hover on the link

    • Optional displayText: string

      Optional display text for the link. If specified, the display text of link will be replaced with this text. If not specified and there wasn't a link, the link url will be used as display text.

    Returns void

    Set background color at current selection

      One of two options: The color string, can be any of the predefined color names (e.g, 'red') or hexadecimal color string (e.g, '#FF0000') or rgb value (e.g, 'rgb(255, 0, 0)') supported by browser.
