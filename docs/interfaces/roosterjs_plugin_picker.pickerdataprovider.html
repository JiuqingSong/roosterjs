onInitalize: function
  • Defined in roosterjs-plugin-picker/lib/PickerDataProvider.ts:59

Function called when the plugin is intialized to register two callbacks with the data provider and a reference to the Editor. The first is called in order to "commit" a new element to the editor body that isn't handled automatically by the editor plugin. The second sets the isSuggesting value for situations wherethe UX needs to manipulate the suggesting state that's otherwise plugin managed.
